## 💡 Запуск проектов по технологиям

### Python 🐍
```bash
# Просто запустите нужный скрипт
python3 Fundamentals-of-Software-Engineering/Решения\ задач/Lab_2/A1.py

# Или для проектов с зависимостями
cd Algorithms-and-Structure-of-Data/lab_1
python3 algorithms.py
```

### C++ ⚙️
```bash
# Option 1: CMake (рекомендуется)
mkdir -p build && cd build
cmake ..
cmake --build .
./lab_name

# Option 2: Одиночные файлы через g++
cd Fundamentals-of-Algorithmization-and-Programming/Решения\ задач/lab_3
g++ -std=c++17 lab_3_2.cpp -o lab_3_2
./lab_3_2
```

### Assembly 💾
Для запуска ассемблерных программ требуется DOS-окружение (DOSBox или виртуальная машина с FreeDOS).

#### Запуск в DOSBox:
```text
dosbox

# Внутри DOSBox:
mount c /home/your-username/Zhdanovich-education
c:
cd Assembly\Решения\ задач

# Сборка и запуск (если TASM/TLINK установлены)
tasm LAB1.ASM
tlink LAB1
LAB1.EXE
```

> [!NOTE]
> Если TASM/MASM не установлены в вашем DOS-образе, вы можете изучать исходники или добавить компилятор вручную.
